# 🎓 Grade Management System

## 📌 Project Overview
The Grade Management System is a Java console-based application that allows users to manage student marks efficiently. It stores marks for multiple subjects, calculates averages, assigns grades, and generates detailed performance reports.

This project demonstrates core Java concepts such as arrays, loops, control flow, and object-oriented programming.

---

## 🚀 Features
- ➕ Add student marks for 5 subjects
- 📋 View all student records in table format
- 📊 Calculate student averages and grades
- 🏆 Find top performers (ranking system)
- 📈 Generate performance report including:
  - Subject-wise averages
  - Top performers
  - Grade distribution
- ✅ Input validation using try-catch

---

## 🛠️ Technologies Used
- Java (Core Java)
- OOP (Object-Oriented Programming)
- Arrays & ArrayList
- Scanner Class (User Input)

## ⚙️ Setup Instructions

Install JDK (Java Development Kit)

### 2. Compile the Project

### 3. Run the Program
java -cp src GradeManagementSystem
